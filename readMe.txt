INSTALLATIE:
Voordat je de applicatie kunt runnen heb je de volgende
libraries nodig:

colorama==0.4.6
contourpy==1.1.1
cycler==0.12.1
DateTime==5.2
fonttools==4.43.1
kiwisolver==1.4.5
matplotlib==3.8.0
numpy==1.26.1
packaging==23.2
Pillow==10.1.0
pyparsing==3.1.1
python-dateutil==2.8.2
pytz==2023.3.post1
six==1.16.0
tabulate==0.9.0
tqdm==4.66.1
zope.interface==6.1

Deze kun je downloaden met instaldependendencies.bat of de
libraries handmatig te installeren.
"pip install -r requirements.txt"

Als je alle libraries hebt geinstalleerd kun je de applicatie runnen.
Zorg ervoor dat gassen.csv, inspecteurs.txt, bedrijven.txt en 
bezoeksrapporten.txt zich bevinden in dezelfde map als alle .py bestanden. 
Verander verder niks aan de bestandsopbouw of aan de code.
=============================================================
LETOP: Deze applicatie is op windows 10/11 gebasseerd. Functionaliteit
is niet getest op linux of macOS.
RUNNEN:
Om de applicatie te runnen moet je dit via de CMD doen, gebruik hiervoor run.bat
om de applicatie geautomatiseerd te laten runnen. Als je dit handmatig wilt doen 
kan je in de terminal van CMD: "python start.py" uitvoeren op de applicatie te runnen.
